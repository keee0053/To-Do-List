# To-Do List

JavaScriptで作成したToDoリストアプリです。  
タスクを追加し、期限を設定すると、残り時間をリアルタイムでカウントダウンできます。

## 公開URL

https://keee0053.github.io/To-Do-List/

## 機能

タスクを追加できる
タスク名を入力できる
期限を設定できる
期限までの残り時間をリアルタイムで表示できる
タスク名と期限を編集できる
完了したタスクを削除できる

## 使い方

1. `+` ボタンを押してタスク行を追加します。
2. タイトル欄にタスク名を入力します。
3. Enterキーを押すと、入力したタスク名が保存されます。
4. 残り時間の欄で期限を選択します。
5. 期限までの残り時間がカウントダウン表示されます。
6. `...` ボタンを押すと、タスク名と期限を編集できます。
7. 完了チェックを押すと、そのタスクを削除できます。

## 工夫したところ

JavaScriptでHTML要素を作成し、ボタンを押すと新しいタスク行が追加されるようにしました。
`datetime-local` を使って、期限の日付と時間を入力できるようにしました。
`setInterval` を使って、残り時間を1秒ごとに更新するようにしました。
タスクごとにタイマーを管理し、編集時には一度タイマーを止めてから再設定できるようにしました。
完了チェックを押すと、対象の行だけが削除されるようにしました。
